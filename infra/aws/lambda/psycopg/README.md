# Creating the Psycopg Lambda Layer (AL2023 / Python 3.12)

This guide walks through building a Lambda layer containing **Psycopg** binaries compatible with **Python 3.12 Lambdas on Amazon Linux 2023**.

## Prerequisites

- **Docker** installed on your machine (used to match the Lambda environment)

## Create ZIP

- Run `scripts/psycopg_zip_create.sh`
